# Data Storytelling App – Global Development Analysis

## Project Overview

This project is a data storytelling and visualization application built using **Python**, **Pandas**, and **Plotly**.

The application analyzes global development indicators using the Gapminder dataset and presents insights through interactive visualizations.

The analysis focuses on:

* GDP per Capita
* Life Expectancy
* Population
* Continental Comparisons
* Global Development Trends

---

## Features

### Data Exploration

* Loads Gapminder dataset
* Analyzes 2007 global development data
* Generates descriptive statistics
* Identifies missing values

### Insights Generated

* Top 5 countries by GDP
* Bottom 5 countries by Life Expectancy
* Population distribution by continent
* GDP and Life Expectancy relationships

### Interactive Visualizations

1. GDP vs Life Expectancy Scatter Plot
2. Life Expectancy Trend Analysis
3. Population by Continent Bar Chart
4. GDP Distribution by Continent Box Plot
5. World Life Expectancy Map
6. Correlation Heatmap

### Storytelling Insights

* Economic growth vs health outcomes
* Population distribution trends
* Regional development comparisons
* Global inequality observations
